Contribute
As RELOAD is Open Source, anyone who wants to can contribute to the project. The code is available from: http://www.sourceforge.net/projects/x4l-reload/. We are currently preparing some developer documentation which will be of use to anyone wishing to integrate their own code. The developer documentation will be distributed through the RELOAD sourceforge pages as well.

Other ways to contribute
Even if you can't program, you can contribute to the project by using the tools and providing feedback in the form of ideas, requests, bugs, anomalies and examples. If you have anthing which you feel may be useful to the community as a whole then please let us know. Send an email to Colin Milligan at colin.mlligan@strath.ac.uk
