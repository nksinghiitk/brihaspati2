Course Management

This is the main page for navigating through the course. Currently there are 3 links i.e. Course Content, Course Configuration and Upload Course Content.

    1. Course Content

    The Course Content link provides link for browsing through the course content. If the user is an instructor in the course, he gets additional links for editing and publishing content material.

    2. Upload Content

    This is a link to the content upload page. Only instructors in a particular course have access to this page.  The instructor can upload multiple files at a time.

    3. Course Configuration

    This page contains configuration options for the particular course in which the user in logged in. Currently there is an option to enable or disable guest login. The instructor can choose to give guests access to his content material by clicking on a buttyon.
