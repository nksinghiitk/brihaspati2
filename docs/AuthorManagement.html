AuthorHome

This is the main page for navigating through the author. Currently there are 3 links i.e. UploadContent,ViewContent and RepositoryBrowser.
1.UploadContent

This is a link for the uploadContent page. Only author have to access  the page.  The author can upload multiple files at a time.
2.View Content

The View Content link provides  for browsing through the available contents of  the author. Author have additional links suchas Delete,Move,View
and Permission of content material.

3.Repository Browser

This page contains list of all author.By using browsing author can see all  the directory and files of all author, who registered as a author.
