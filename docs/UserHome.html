User Home Page

This is the first screen in the sequence of screens which the user(instructor or student) in a course
first sees when he logs in to the system. This screen lists out the courses in which a user is registered
and what is is his role in the course i.e. student or instructor.

The screen also contains information about messages in a particular course i.e. it  notifies the user about
course specific messages. Course specific navigation starts after the user enters a particular course.
The user can return to this page from anywhere inside the course by clicking on Home link.
