To save Chat Text, Canvas and White Board
You need to make some changes in java Policy File

STEP 1:Go to /usr/java/jdk1.5.0_04/jre/lib/security/
STEP 2: Open the file "java.policy" and add these given below line :
permission java.io.FilePermission "${user.home}${/}*", "read,write";
permission java.io.FilePermission "${user.home}${/}-", "read,write";
permission java.io.FilePermission "${user.home}${/}", "read,write";
permission java.io.FilePermission "{user.home}/tdk-2.3_01/webapps/brihaspati2/babylon/babylon.jar" ,"read,write";
