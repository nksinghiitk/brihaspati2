How to make a different Group

 

1. Usage Group – Under this instructor can create single group and choose the students you want to add to it.

  2. Multiple Group - Create empty groups to which you can add students later or create groups in which students are randomly distributed.

  3. Signup Group - Create empty groups and allow students to select the groups they want to join by using sign-up forms.

  Note : Creating a multiple group and signup group please check the avilable no of student
  than fill the (number of student per group)student no field according to avilable students and groups.
