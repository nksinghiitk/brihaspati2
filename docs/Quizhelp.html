Quiz Management
In quiz management the insructor can take the quiz of own course.
first time there display the Quiz Scheduling.after quiz scheduling there display the quiz detail.
when student submit the quiz,after ending the quiz time instructor give the grade(check the quiz if quiz type is short ).if quiz type is True/False or Multiple choice,the grade of student automatically generated.
1.Quiz Schedule
In quiz scheduling the instructor scheduled the quiz i.e.(about date,time and quiz title)
2.Quiz Detail
This page contains Create Quiz , announce , View ,Delete and Update the Quiz Schedule.
in Crate Quiz the instructor create the question. if any one question create in that quiz, AddQuestion replaces Create Quiz
in Announce instructor announce the quiz to the student of that course.
if instructor wants to change the question and correct answer go to the View for editing the question and answer.
Instructor can delete a single quiz or more than one at a time.
Instructor can update the quiz date,time and maxmarks. 3. Quiz Grade
In this the instructor check the quiz(give the marks).if question is short type.
4.Grade Card
In grade card instructor can see the grade card of particular student and also can update the total marks of student
