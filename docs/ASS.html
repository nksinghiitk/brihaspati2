ASSIGNMENT

This is the main page for assignment. Currently there are 4 links i.e. Post Assignment, View Assignment, Post Answer, Post Grade, Repost Permission.


1. Post Assignment
The Post Assignment link provides link for uploading the Assignment . Only the Instructor has the right to upload the assignment. (Instructor gives the assignment to the student which have Topic name, Due date and Grade etc. ).

2. View Assignment
This is a link to view the assignment. Instructors and Students both have an access to this page. For viewing the Assignment, we have to select the Topic name.
   
3. Post Answer
This page contains Topic name and Attach file option.txt, .pdf and .html

4. Repost Permission
This link have Student name, Topic name and Due date. Instructor can select a specific student or can also be select all the students by the select option.
