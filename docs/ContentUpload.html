Uploading Content

The Author can upload content material in his Repository. The author can choose to upload content for an existing topic or a new topic.               
There is a drop down list containing list of topics in the particular directory. The author can choose one existing topic while uploading content.                      
He can also create a new topic there itself by selecting new toipc. In this case he should give a valid topic name. The name of the topics inside                          
a course should be unique.There is fascility for multiple file upload i.e.the author has the option of uploading more than one files in one go.                            
This helps him in saving time.
