View Online Registration

This is the administrator approval page where the admin can approve the Online Registration. there will be two link one is view user Registration and second one is view Course Registration. click on the (View User Registration) Online User Registration page will be open, when click on (View Course Registration) Online Course Registration page will be open.

Accept Button: select user or course and click on the accept button the user or course become active and selected user or course deleted form the page. Mail will be reached to him according his Email-ID.

Reject Button: select the user or course and click on the reject button the selected user or course will be deleted from the page.
