Research RepositoryGroup

This is the main page for Research Repository Group. Currently there are two links i.e. Create Research Repository and Research Repository Groups .

    1. Create Research Repository

    The Create Research Repository link provides link for creating Research Groups. To create a new Research Groups click on Research Repository link a new window will appear. Enter Rsh group name and description then click on create button. on clicking a message that (Research Group created Successfully !! !!) will be shown.
     
    2. Research Repository Groups
  
    This is a link to show all Existing Research Groups.
   
    There are number of four Available Actions in this page.

    (i) Join

    To join any Group just click on Join action.(If user wants to post any topic in any group he/she have to join that group).
     
    (ii) Unjoin

    Under this action only Rsh owner or admin can work.

    (iii) View

    Under this action any user can view topics in any group.

    (iv) Post

    To post any topic click on Post action a new window will appear. Enter Subject and write message then click on AddNewTopic button. On clicking a message(Send Successfully !!) will be shown.
     

    This page also provide search facility.On this page search is available by two way.

    (i) By Author Name

    Type Author Name in match string text box then click on search button it will display all the Research Group related to that user.
     
    (ii) By Research Group

    Just type the first character of the group and then single click on search button will display all the research group started with that character.
