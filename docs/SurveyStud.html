SURVEY BY STUDENT
When student entered in his course.there is a link in Course Home, if survey is created of that course. The link will be hidden when student submit the survey of that Course.
If Survey is available then a screen contain the Course name with Question having option ( Very Poor, poor, Normal, Good, Very Good ). The student select the option to surveying for that Course.
The student post the survey. After posting the survey the button "post" is unactivate for that course. Because student can give only once marks for one course.
