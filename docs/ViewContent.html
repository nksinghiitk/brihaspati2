View Content

The View Content link provides link for browsing through the available contents of author. author gets additional links for
Delete,Move,View and Permission of content material.

Delete

The author can delete his own directory and files.

Permission

The author can give permission(content topic) to any author, instructor(Private area) and instructor(Course area).
View

The author can view his file(html,txt,gif & tex).
Move

The author can move file one directory to another directory
