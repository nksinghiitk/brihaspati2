Group Management

This is the main page for groupmanagement. Currently there are 5 links i.e. Creategroup, Add/View Member,View/Delete Group, Edit/Remove Member and Activity.

    1. Create group

    The Create group link provides link for creating group. Instrucror can create three types of groups (Usage group,Multiplegroup and Signup group).
     
    2. Add/View Member
  
    This is a link to the Add/View Member. Only instructors in a particular group have access to this page. The instructor can add multiple members at a time.
   
    3. View/Delete Group
   
    This page contains View/Delete Group options for the particular group in which the user in logged in. Delete his own groups.
   
    4. Edit/Remove Member

    Under this link Instructor can edit the group by Edit group link  and remove member of particular group by Remove member link.
   
    5. Activity

   Students can work collaboratively and privately in their groups using the mail, discussion,Notices and chat tools.
   The instructor can also create group assignments and assignment them to groups. Groups are also useful to separate multiple sections of a course.
